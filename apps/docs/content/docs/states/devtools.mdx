---
title: DevTools
---

Redux DevTools integration provides powerful debugging capabilities for your store. With DevTools enabled, you can inspect state changes, time-travel through actions, and debug your application's state management in real-time.

```tsx
import { createStore } from 'finalstore';

const store = createStore({
  states: {
    count: 0
  } as {
    count: number;
  },
  actions: {
    increment: (state) => {
      state.count++;
    }
  },
  config: {
    name: 'MyStore', // Name in DevTools
    devtools: true // Enable DevTools
  }
});
```

Features available in DevTools:

- Time-travel debugging
- Action history
- State inspection
- Import/Export state
